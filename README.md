# AI Interview Evaluation System

## 📌 Project Overview
The AI Interview Evaluation System is a web-based application that helps users practice mock interviews using AI. It evaluates answers and provides feedback, scores, strengths, and improvement suggestions.

---

## 🚀 Features
- User Authentication (Firebase)
- Resume Upload (PDF)
- AI-based Question Generation (Gemini API)
- Real-time Interview Chat
- Final Evaluation Report (Score, Feedback)
- Interview History Tracking

---

## 🛠 Tech Stack

### Frontend
- React.js
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Authentication
- Firebase Authentication

### AI Integration
- Google Gemini API
